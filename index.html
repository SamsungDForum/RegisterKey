Register Keys Example

Available buttons/actions:

Up - move select box up
Down - move select box down
Enter - toggle registration of selected key
  • Red
  • Green
  • Yellow
  • Blue
Logs:

Info

This application demonstrates the process of registering and unregistering the keys handled by application

When the application is started out of color keys only the red key is registered. This means pressing these keys on remote does not send events to the application.

Some keys (arrows, volume up/down, channel up/down, enter, exit etc...) are always registered.
